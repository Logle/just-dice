# just-dice simulator

simulates a bunch of rolls using the algorithm at Just-Dice and plots them

## how to use

simulate rolls: `node rolls.js`

plot rolls: `python plot.py`

## example simulation

data: http://dinosaur.io/media/projects/just-dice/rolls.zip

```
$ cat rolls.csv | wc -l                                       
27341150
```
